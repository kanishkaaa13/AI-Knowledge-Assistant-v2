# AI Knowledge Assistant v2 Roadmap

This roadmap outlines the planned evolutionary phases of the AI Knowledge Assistant, transitioning the system from a standard Retrieval-Augmented Generation (RAG) platform to an advanced Agentic Knowledge Graph workspace.

---

## 🚀 Evolutionary Phases

### ⏹ Phase 1: Core Platform (Done)
* **Vector-based RAG:** Document chunking, vector embedding generation, and semantic similarity search.
* **Persistent Conversation Memory:** Context-aware chat sessions.
* **Smart Summaries & Quizzes:** Dynamic generation of summaries and quiz questions from documents.
* **Dashboard Analytics:** High-performance dashboard with usage metrics and visualizations.

---

### ⏳ Phase 2: OKF Foundation (Planned)
* **OKF Schema Design:** Standardize the Open Knowledge Format schemas for concept representation.
* **Concept Extraction:** Implement automated parser logic to distill key concepts, properties, and definitions.
* **Metadata Association:** Link extracted concepts back to source document chunks.
* **OKF UI Explorer:** Build frontend views to view and search extracted concepts.

---

### ⏳ Phase 3: Knowledge Graph (Planned)
* **Relationship Discovery:** Design algorithms to identify links and relationships between extracted concepts.
* **Graph Storage & Indexing:** Integrate a graph store or graph-based indexing alongside the vector store.
* **Graph Visualization:** Create interactive 2D/3D graph visualization components in the frontend.
* **Associative Search:** Query the graph for connected concepts and documents.

---

### ⏳ Phase 4: Agents (Planned)
* **Agent Orchestration Loop:** Build execution loops for goal-directed agents.
* **Tool Integration:** Equip agents with tools for web searching, document retrieval, and math computation.
* **Agent UI Dashboard:** Show agent execution steps, thought process, and final answers in a unified view.

---

### ⏳ Phase 5: Advanced Reasoning (Planned)
* **Hybrid Search (Graph + Vector):** Integrate advanced search combining semantic vector distance with graph walks.
* **Multi-hop QA:** Query across multiple documents and relations to answer complex questions.
* **Self-Correction & Memory:** Let agents automatically review and correct their plans based on output evaluation.
